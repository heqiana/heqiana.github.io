项目打包

项目目录

app/ 项目的主要代码
dist/ 项目部署上线文件
public/ 静态文件，比如css，图片，字体。
tests/ 单元测试和集成测试
tmp/ 项目缓存，比如debug的输出文件。
bower_components/ Bower 依赖
node_modules/ npm 依赖
vendor/ 外部依赖，npm和bower不能安装的
.jshintrc JSHint 配置文件
.gitignore Git 忽略的配置文件
ember-cli-build.js 项目编译信息。比如less编译、babel转码、jshint是否开启、js压缩、css压缩、项目依赖资源导入（比如bower_components中的组件，vendor中的组件和全局js等）。
bower.json bower依赖列表
package.json npm 配置依赖列表
